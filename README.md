# 🏆 Lucas Silva Official

> Portfólio digital com e-commerce integrado do atleta fictício Lucas Silva.

Projeto acadêmico full-stack desenvolvido em **Java + Spring Boot + MySQL**, com identidade visual profissional inspirada em sites oficiais de atletas como Cristiano Ronaldo e Lando Norris.

---

## 🚀 Funcionalidades

### 📰 Portfólio do atleta
- Home com hero, números animados e notícias em destaque
- Página "Sobre" com biografia, trajetória e conquistas
- Estatísticas com **3 gráficos interativos** (Chart.js)
- Galeria com filtros por categoria (partidas, treinos, comemorações)
- CRUD completo de notícias com persistência no MySQL
- Formulário de contato com validação dupla (cliente + servidor)

### 🛒 E-commerce integrado
- Página de produto do livro autobiográfico
- Preço com desconto, avaliações, selos de confiança
- **Checkout simulado** com 3 métodos de pagamento:
  - **PIX** — QR Code visual, código copia-e-cola e contagem regressiva
  - **Cartão de Crédito** — máscara de digitação e parcelamento em até 12x
  - **Cartão de Débito** — pagamento à vista
- Tela de confirmação com número de pedido aleatório
